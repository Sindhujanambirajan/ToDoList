#  TODO App

A lightweight and user-friendly client-side TODO list application built using HTML, CSS, and JavaScript. This application allows users to manage daily tasks efficiently with persistent storage using the browser's localStorage.

##  Features

* Add new tasks using the input field and **ADD** button
* Delete tasks using the **Delete** button
* Mark tasks as completed by clicking on the task
* Completed tasks are indicated with a tick icon
* Task data is stored using **localStorage**, so tasks remain available even after refreshing the page

## Project Structure

* `index.html` — Main application page and task functionality
* `style.css` — Styling for the application
* `tick.jpg` — Tick icon displayed for completed tasks

##  Usage

1. Open `index.html` in your web browser.
2. Enter a task in the input field.
3. Click **ADD** to add the task to the list.
4. Click a task to mark it as completed.
5. Click **Delete** to remove a task.

##  Notes

* Tasks are stored locally in the browser using `localStorage`.
* This is a front-end project and does not include a back-end or database.

##  Technologies Used

* HTML5
* CSS3
* JavaScript
